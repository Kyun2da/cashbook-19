### 기능 목록

- 로그인하기 전에는 더미 데이터 출력
- 월별 수입/지출 리스트
- 수입/지출 추가, 삭제
- 카테고리, 결제수단 추가, 삭제
- 월별 수입/지출 요약 달력 제공
- 월별 지출 요약 그레프 제공
- github 간편 로그인 기능
- 모바일, 테블릿, PC 모두 사용가능한 반응형웹!
